# MONEVSRL

Clean GitHub version of the MONEVSRL web application.

## Files
- `index.html`
- `style.css`
- `app.js`
- `config.js`
- `netlify.toml`
- `supabase-schema-fixed.sql`

## Setup
1. Create a Supabase project.
2. Run `supabase-schema-fixed.sql` in Supabase SQL Editor.
3. Fill `config.js` with your Supabase Project URL and anon public key.
4. Deploy the folder to Netlify.

## Notes
- Public registration is for student users only.
- Mentor accounts are created manually by admin in Supabase.
